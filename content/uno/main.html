This is a text-based UNO program written in Python.
It was written for a final project in one of my college classes.
It was made in 2 weeks, and although it's complete, I plan on eventually adding more features.

This program only requires Python to be installed to run.
It does not require any dependancies, as it only uses built-in libraries.

Features:
  • A menu to change different settings and aspects of the game
  • Modern UNO rules like jump-ins, stacking, challenging, etc.
  • Multiplayer (all players play on 1 device, this script doesn't use networking stuff)
  • Change the amount of players per game (up to 16)
  • Extra card colors (purple, orange, black, and white)
  • Teams
